# 🚀 Composio API Research Dashboard

A modern AI-powered dashboard for analyzing and visualizing research on 100+ APIs. This project provides an interactive interface to monitor API verification status, authentication methods, analytics, and AI-generated insights.

Link : https://composio-api-research-dashboard-d869-cthb5nccg-lokes3092.vercel.app/#insights

---

## 📌 Overview

The Composio API Research Dashboard was built to simplify the analysis of APIs collected during research. Instead of reading hundreds of documentation files manually, users can quickly view statistics, charts, and AI-generated insights from a single dashboard.

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Lokesh3092/composio-api-research-dashboard.git
```

Go to the project folder

```bash
cd composio-api-research-dashboard
```

Open the project in VS Code.

Run Live Server or open:

```
html/index_new.html
```
